## Install the projects

```bash
docker-compose up -d
```

Connect
```bash
docker exec -it rust /bin/sh
```

On container
```bash
cargo new grrs
```


$ cargo new grrs
     Created binary (application) `grrs` package
$ cd grrs/
$ cargo run
