# Vue JS


## Installer

```bash
npm install -g @vue/cli
npm install -g @vue/cli-init
vue init vuetifyjs/webpack vuetify-tutorial
cd vuetify-tutorial
npm run dev
```

##  Run new project
```bash
npm install -g @vue/cli
```
OR
```bash
yarn global add @vue/cli
```

Create a project:
```bash
vue create my-project
```
OR
```bash
vue ui
```
